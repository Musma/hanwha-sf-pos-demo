# Prototype pages

이 폴더에는 통합 `index.html`을 생성할 때 사용하는 기존 SF-POS 화면 원본과 DataCanvas 런타임을 보관한다.

- `의장 주간작업계획 수립.html`: 통합 문서의 기본 번들
- `워크프론트 점검 메인.html`: 워크프론트 메인 화면 원본
- `워크프론트 점검 서브.html`: 워크프론트 상세 화면 원본
- `*.dc.html`, `Header.dc.html`, `SideNav.dc.html`: 편집용 화면 소스
- `support.js`: 편집용 화면 런타임

배포할 때는 이 폴더가 필요 없다. 저장소 루트의 `index.html` 한 파일만 호스팅하면 된다.
